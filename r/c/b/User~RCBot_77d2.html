RCBot is an interlingually operating bot operated by Richie to help with issues related to the Wikimedia Commons. As of now, the bot is still in development and testing, so please report all encountered issues and other feedback to its talk page on the Commons. Please note that the talk page of this account is not read, use the one on the Commons). -- RCBot 19:41, 16 Apr 2005 (UTC)
